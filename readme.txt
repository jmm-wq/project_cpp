SpaceFactory - Gra w c++ stworzona z użyciem SFML

Build
-----

- Install [SFML 2.5.1](https://www.sfml-dev.org/download/sfml/2.5.1/)
- Pliki źródłowe obejmują projekt w Code::Blocks
- Przesłana wersja (ostateczna) projektu nosi nazwę SpaceFactoryCopy
- Sugerowany build: import projektu do Code::Blocks   (.cbs)
